

# english-anagrams

> Find anagrams for English words. Works offline.

## Installation

```sh
npm install english-anagrams
```

## Usage

```js
const anagrams = require('english-anagrams')

anagrams('nerdish')
// => [ 'hinders', 'nerdish', 'shrined' ]
```

Note: Search is case *insensitive*
